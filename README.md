A repository is a location where we can keep and organize the files for our specific projects, such as code, images, documentation etc. In GitHub, repositories are essential to project organization and version control. They offer so many other benefits which make it more user friendly.For ex- tracking changes in the project, collaboration with others, and preserving project history.
The advantages of Github repository are as follows -
* We can control the version according to our requirement
* Easy to create Branches within a repository and can do experiment on the code without affecting the maincode base
* Issues and Pull request is possible .Hence help in making the code more efficient
* We can make our repository public or private
* Atlast README file are added in the repository which provides information about overall project setup
